HTML Tables

Tag Name Definition
<table> Table Creates a blank table (No rows or columns)
<tr> Table Row Creates a blank row
<td> Table Data This is where you write the content of a cell
<thead> Table Head Used to separate the head of the table from the rest
<th> Table Head Used within the <thead> as column headers
<tbody> Table Body Used to separate the body of the table from the head
<tfoot> Table Footer Used to separate the footer of the table from the rest

HTML Table Attributes

Attribute Name Definition
<colspan> Column Span Defines how many columns a <td> element should span
<rowspan> Row Span Defines how many rows a <td> element should span

Common HTML Tags

Attribute Name Definition
<!DOCTYPE html> Doctype Lets the browser know that it should interpret the code as HTML
<div> Division Acts as another container for additional elements (not-semantically)
<p> Paragraph The place to put more text
<head> Head Used to contain all of the head (title, style, meta, etc.)
<title> title Sets the browser's title (in the tab)
<link> link/td> Defines the relationship between the current document and an external source
<style> Style Used to write CSS code in the HTML (Not commonly used/suggested)
<span> Span An inline HTML container used to assign the class attribute for styling purposes
<h1> Heading Can use <h1>through<h6>as headings in a page
<em> Emphasized text Used semantically inline for screen readers and styling
<b> Bold Styles text as bold and for screen readers
<i> Italic Styles text as italic and for screen readers
<u> Underline Styles text as underlined and for screen readers
<a> Anchor Used to link one page to another
<ol> Ordered List Creates a numbered list
<ul> Unordered List Creates a bulleted list
<li> List Item Used inside the Un/Ordered list to write items
<!--Statement --> Notes? Used to leave notes in HTML code
<br> Break Used inline in a paragraph to create a line break
<img> Image Used to include an image (self-closing)
<video> Video Must specify source, and controls attribute
<audio> Audio Must specify source, controls attribute
<embed> Embed Self-closing tag to implement any type of media

Semantic HTML Tags

Attribute Definition
<header> Specifies a header for a document or section
<nav> Defines navigation links
<section> Defines a section in a document
<article> Defines independent, self-contained content
<aside> Defines condent aside from the page content
<footer> Defines a footer for a document or section
<figure> Specifies self contained content, like illustrations, diagrams, photos, code listings, etc.
<figcaption> Defines a caption for <figure> element
<details> Defines additional details that the user can view or hide
<main> Specifies the main content of a document
<mark> Defines marked/highlighted text
<summary> Defines a visible heading for a <details>element
<time> Defines a date/time

HTML Attributes

Attribute Name Definition
<href> Hyperlink Reference Used with an <a>tag and specifies the URL of the page the link goes to
<src> Source Used with an <img>tag to embed an image
<width> Defines the width of an image (in pixels)
<height> Defines the height of an image (in pixels)
<alt> Alternative Defines alternate text for an image, if the image cannot be displayed
<lang> Language Used inside<html>tag to declare language of Web page
<title> Title Defines extra information about an element
<id> ID Specifies a unique ID for an HTML element (must be unique within the HTML document
<target> Target Used in an anchor tag to specify where a hyperlink opens. A value of<"_blank"> opens the in a new tab
<#> Link to a Different Part of the Page Used within an anchor tag to create hyperlinks to a different part of the same page

CSS Basics

Name Definition
Class Selector Denoted with a. and followed by the class name
ID Selector Denoted with a# and followed by the id name
!important Overrides any other declarations and ignores other specificity. (Don't use it, its bad practice)
Type Selectors Used to match all elements of a given type or tag name
CSS Declarations
font-size Sets text size, many different units or types, such as px
background-color Contorls background color of elements
opacity Can be used to control transparency, ranges from 0-1
font-weight Contorls boldness of text, can be keyword such as bold or normal
text-align Sets text alignment of inline contents
color Sets the color for an element
font-family Sets the typeface, wrap in quotes
background-image Sets the background image, should be a url in this syntax:url("nyan-cat.gif")
border Sets the border (for a table)
border-collapse Set to collapse to get rid of the spaces between table elements
padding Sets the padding around table data
